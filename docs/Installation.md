# Инструкция по установке кастомной активности

Выполните следующие простые шаги, чтобы добавить активность «Соединить две таблицы» в вашу PIX Studio.

---

### Шаг 1. Скачайте файл активности
1. Перейдите на страницу проекта на **GitHub**.
2. Откройте вкладку **Actions** в верхнем меню.
3. Выберите последний успешный запуск сборки (Workflow Run).
4. В самом низу страницы в разделе **Artifacts** (Артефакты) нажмите на архив, чтобы скачать его.

### Шаг 2. Извлеките библиотеку
1. Распакуйте скачанный ZIP-архив.
2. Найдите внутри файл с расширением `.dll`: `Activities.Custom.dll`.

### Шаг 3. Скопируйте файл в папку PIX Studio
1. Откройте проводник и перейдите в корневую папку установленной PIX Studio. 
   По умолчанию она находится по пути:
   `C:\Program Files\PIX`
2. Скопируйте файл `Activities.Custom.dll` и вставьте его **прямо в эту папку** (рядом с исполняемым файлом `Studio.exe`).

> ⚠️ **Обратите внимание:** Для копирования файлов в папку `Program Files` операционная система Windows запросит права администратора.

### Шаг 4. Проверьте результат в интерфейсе
1. Если PIX Studio была открыта, **перезапустите её**.
2. В левой панели «Активности» разверните группу **Custom activities**.
3. Вы увидите новую активность **Соединить две таблицы**. Перетащите её на рабочую область для начала использования!
